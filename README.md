# Love-Maths
## Love Maths - JavaScript Walkthrough Project

Love Maths is a fully responsive JavaScript-based maths game, created as part of the Code Institute JavaScript Essentials course walkthrough. This project demonstrates how to use pure JavaScript in a real-world context, focusing on DOM manipulation, event handling, and functions.

## Features

- Interactive maths game with addition, subtraction, multiplication, and (challenge) division modes.

- User-friendly interface with responsive design.

- Focus on JavaScript functionality rather than complex HTML/CSS styling.

- Opportunity to practice writing your own division game as part of the challenge.

## Learning Outcomes

- Understand how JavaScript interacts with the DOM.

- Implement event listeners to handle user input.

- Avoid using global variables by structuring code efficiently.

- Gain confidence in writing JavaScript functions in a real-world project.

## Usage

- Clone the repository, open index.html in your browser, and start playing. Follow the walkthrough to build your own version or extend the game functionality.
